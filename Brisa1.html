Ciberseguridad 01

Tipos de Malware

Tipo de Malware Función Impacto
Virus Autoreplicable, infecta arhivos Daño de datos al igual que de sistema
Gusano Se propaga a traves de las redes Interrupción de la red, agotamiento de recursos
Caballo de troya Se disfraza de software legitimo Robo de datos, control de sistema
Ransomware Cifra archivos y exige el pago Robo de datos, control del sistema
Software espia Supervisa en secreto la atividad del usuario Valoracion de privacidad, robo de datos
Software publicitario (Adware) Muestra anuncios no deseados Molestia, de gradación del rendimiento
Rootkit Ocultar software malicioso Acceso no autorizado, amenazas persistentes
Keylogger Registra las pulsaciones de teclas Robo de credenciales, filtración de datos
Fileless Malware
Malware sin archivos		 Corre de la memoria Ataques sigilosos y dificiles de detectar
Cryptojacker Minas de criptomonedas Problemas de rendimiento, secuestro de recursos
Red de boots (Botnet) Crea una red de dispositivos Ataque, distribución de spam
Bomba Lógica (logic Bomb) Ejecuta codigo en el disparador Destrucción de datos, daños al sistema
